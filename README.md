# yeeway's Skills

個人整理的 [Agent Skills](https://agentskills.io/home) 集合，可搭配 [`npx skills` ](https://github.com/vercel-labs/skills) CLI 來管理與使用。


## 目錄

| Skill | 說明 |
| --- | --- |
| [article-summarizer](article-summarizer/SKILL.md) | 文章摘要、TL;DR、讀連結；支援 URL 抓取與貼上全文，輸出繁中並保留語氣與結構。 |
| [code-review-react-ts](code-review-react-ts/SKILL.md) | TypeScript React code review 指引（型別、React、效能、安全、架構、無障礙、錯誤處理、測試）|
| [web-interface-guidelines](web-interface-guidelines/SKILL.md) | 對照 [raunofreiberg/interfaces](https://github.com/raunofreiberg/interfaces) 的網頁介面實務準則（互動、排版、無障礙等）。 |
